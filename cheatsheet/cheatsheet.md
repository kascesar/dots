# Cheatsheet

## GNOME

### Ventanas y workspaces

| Atajo | Acción |
|-------|--------|
| `Super+Tab` | Cambiar entre aplicaciones |
| `Super+`` ` | Cambiar entre ventanas de la misma app |
| `Super+↑` | Maximizar ventana |
| `Super+↓` | Restaurar / minimizar ventana |
| `Super+H` / `Super+L` | Enfocar ventana izquierda / derecha (Pop!_Shell) |
| `Super+Shift+↑↓←→` | Mover ventana entre workspaces/monitores |
| `Super+PgUp/PgDn` | Cambiar workspace |
| `Super+Return` | Entrar al modo tile (Pop!_Shell) |
| `Super+G` | Alternar ventana flotante (Pop!_Shell) |
| `Super+Y` | Alternar tiling (Pop!_Shell) |
| `Super+O` | Cambiar orientación de tile (Pop!_Shell) |
| `Super+Q` | Cerrar ventana |

### Atajos custom

| Atajo | Acción |
|-------|--------|
| `Super+T` | Abrir Kitty |
| `Super+H` | Abrir este cheatsheet |

## Yazi

### Navegación

| Atajo | Acción |
|-------|--------|
| `h j k l` | Navegar (←↓↑→) |
| `H` / `L` | Ir a directorio anterior / siguiente (historial) |
| `g g` | Ir al primer archivo |
| `G` | Ir al último archivo |
| `z` | Abrir zoxide (saltar a directorio frecuente) |
| `Enter` | Abrir archivo con app por defecto |

### Selección y operaciones

| Atajo | Acción |
|-------|--------|
| `Space` | Seleccionar / deseleccionar archivo |
| `v` | Modo selección visual |
| `V` | Invertir selección |
| `y` | Yank (copiar) archivo(s) |
| `x` | Cut archivo(s) |
| `p` | Pegar archivo(s) |
| `d` | Mover a la papelera |
| `D` | Eliminar permanentemente |
| `r` | Renombrar archivo |
| `a` | Crear archivo / directorio |

### Búsqueda y filtros

| Atajo | Acción |
|-------|--------|
| `f` | Filtrar archivos visibles |
| `s` | Buscar con fd (nombre) |
| `S` | Buscar con ripgrep (contenido) |
| `/` | Buscar en directorio actual |
| `n` / `N` | Siguiente / anterior resultado |

### Vista y misc

| Atajo | Acción |
|-------|--------|
| `.` | Alternar archivos ocultos |
| `q` | Salir |
| `~` | Abrir configuración de Yazi |
